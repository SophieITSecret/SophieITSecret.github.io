# BARソフィー 画面構成ツリー

更新日：2026-06-15

---

## 凡例

```
→     画面遷移（タップ/クリック）
[条件] アクセス制御（checkAccess）
(js)  担当ファイル
⊕     サブメニュー展開（list-view内）
⊞     全画面オーバーレイ（z-index:99997〜）
◎     カウンター (nav-main) に常駐するボタン
```

## カウンター（nav-main）からの画面遷移

```
nav-main
├── ◎ 🌙今日のアドバイス (pill)
│       → showDailyAdvice()  (fortune.js)
│
├── ◎ お店の使い方 (hint pill) [ログイン済のみ]
│       → guide.showGuideScreen()
│
├── btn-music → ⊕ 音楽メニュー
│       ├── DJソフィーの歌とお酒の物語 → djSophie.showDJList()
│       │       └── DJリスト → 曲選択 → 前口上+YouTube再生
│       └── 音楽リクエスト → music.openMusic()
│               └── ジャンル選択 → 曲一覧 → 曲詳細カード
│                       └── ⊞ 全画面歌詞・解説
│
├── btn-sake → ⊕ お酒メニュー
│       ├── お酒を探す → liquor.openLiquorPortal()
│       │       ├── カテゴリー一覧 → 銘柄リスト → 銘柄カード
│       │       └── スクリーニング → 検索結果 → 銘柄カード
│       └── お酒の話 → music.openTalk()
│               └── テーマ選択 → 話のリスト → 再生
│
├── btn-news → ⊕ NEWS・マーケット
│       ├── テレ朝NEWS24 → YouTube (l-side埋め込み)
│       ├── ウェザーニュース → 外部リンク
│       ├── ライブカメラ → ⊕ 国内/海外/特別
│       │       └── 各カメラ → YouTube (l-side埋め込み)
│       ├── マーケット → ⊕ チャート表示
│       │       ├── 日経平均 / ドル円 / 原油 / S&P500 → TradingViewチャート
│       │       └── コード入力 → チャート or 外部検索
│       ├── ニュース・情報 → ⊕ 外部サイト一覧
│       └── 便利情報 → ⊕ フライトレーダー等 + Wikipedia検索
│
└── btn-notice → favorite.openNotice() [氷音SE付き]
        └── お知らせ一覧
```

## コンソール（btn-grid）Sボタン → ソフィーメニュー

```
S ボタン（handleSButton → showSophieMenu）
├── [画面固有メニュー] (nav.state 別)
│       tit    : DJソフィー（解説+自動再生）/ 連続再生
│       lq_scr : スクリーニング使い方ガイド（準備中）
│       techo  : 連続再生（お気に入りの歌）
│
├── いいお店を探す [checkAccess('restaurant_search')]
│       → restaurant.showRestaurantSearch()
│
├── 🔮 ソフィーの天命診断 [checkAccess('fortune_haiku')]
│       → fortune.showFortuneMenu()  ※後述
│
├── じゃんけん勝負
│       → janken.startJanken()
│
└── この日はどんな日（近日公開・disabled）
```

## 天命診断メニュー（showFortuneMenu）

```
fortune.showFortuneMenu()
├── 🌙 今日のアドバイスを聞く → showDailyAdvice()
│       ├── [入力] ライト/プレミア選択 + 生年月日 + 性別 + 人物帳ボタン
│       │       └── 人物帳 → showPeopleBook()
│       │               └── 人物選択 → showDailyAdvice(onBack, prefill) に戻る
│       ├── [キャッシュあり] 結果表示（本日取得済み） + 引き直しボタン
│       ├── [ライト送信] checkAccess('fortune_haiku') → GAS grade:light
│       ├── [プレミア送信] checkAccess('fortune_sonnet') → GAS grade:premium
│       └── [結果] テキスト表示 + localStorage保存
│               └── 閉じる → onBack()
│
├── 🌟 あなたのご相談 → showMyFortune()
│       ├── [入力] ライト/プレミア選択 + 生年月日 + 性別 + テーマ + 心配事 + 人物帳
│       │       └── 人物帳 → showPeopleBook() → showMyFortune(onBack, prefill)
│       ├── [ライト] テーマ: 総合運/仕事運/恋愛運/金運のみ選択可
│       ├── [プレミア] 全8テーマ選択可
│       ├── 送信 → GAS (grade: 'light'|'premium')
│       └── [結果画面]
│               ├── 📊 命式を確認する（トグル）
│               ├── 📋 結果をコピー
│               ├── ⊞ 📄 診断を全画面 → showResultFullscreen()
│               └── ⊞ 🀄 命式表を全画面 → showMeishikiHtmlFullscreen()
│                       └── ⊞ 📖 命式解説 → showMeishikiGuide()
│
├── 👤 この人どんな人 → showAboutPerson()
│       ├── [入力] ライト/プレミア選択 + 生年月日 + 性別 + テーマ + 気になる点 + 人物帳
│       │       └── 人物帳 → showPeopleBook() → showAboutPerson(onBack, prefill)
│       ├── [ライト] テーマ: 性格分析/SWOT分析のみ選択可
│       ├── [プレミア] 全6テーマ選択可
│       ├── 送信 → GAS (grade: 'light'|'premium')
│       └── [結果画面] showMyFortune と同一構成
│
├── 💑 相性を見てもらう [checkAccess('compatibility')]
│       → showCompatibility()
│       ├── [入力] ライト/プレミア選択（右側縦配置）
│       │         あなた: 生年月日 + 性別 + 人物帳
│       │         お相手: 生年月日 + 性別 + 人物帳
│       │         相手への気持ち + 知りたいこと
│       ├── 送信 → GAS (grade: 'light'|'premium')
│       └── [結果画面] showMyFortune と同一構成
│
└── 👥 人物帳 → showPeopleBook()
        ├── 人物一覧（Firestoreから取得）
        ├── 新規追加 → 入力フォーム
        └── 人物選択 → 呼び出し元画面に prefill で戻る
```
