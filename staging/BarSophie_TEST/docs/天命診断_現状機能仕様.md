## 3. 機能①「あなたのご相談」（showMyFortune）

### 入力フォーム

| 項目 | 要素 |
|-----|------|
| 生年月日 | ft-year / ft-month / ft-day |
| 性別 | ft-gender ボタン（男性・女性） |
| 鑑定テーマ | ft-theme ボタン × 8種類（下記） |
| 心配事（任意） | ft-worry テキスト入力 |

**鑑定テーマ選択肢**
- 🌟 総合運
- 💼 仕事運
- 💕 恋愛運
- 💰 金運
- 📅 向こう1ヶ月
- 📅 向こう1年
- 🔭 向こう10年
- 💬 これが聞きたい

### 結果画面ボタン

| ボタンID | 機能 |
|---------|------|
| ft-meishiki-btn2 | 命式を確認する |
| ft-copy | 結果をコピー |
| ft-fullscreen-result | 診断を全画面表示 |
| ft-fullscreen-meishiki | 命式表を全画面表示 |

### APIに送るプロンプト

```
ソフィーは20代の若い女性バーテンダー。さわやかで知的、品がある。口調は丁寧な「です・ます」調。マダムのような馴れ馴れしさやタメ口は使わない。お客様は40〜50代の紳士が多い。四柱推命の専門用語はさりげなく使い、少し意味を補足するが説明的になりすぎない。
【現在の日付】（実行日付が入る）
※鑑定は必ずこの日付を基準に行うこと。「来年は○○年」などの表現も正確に。
あなたは四柱推命を極めた占い師であり、BARソフィーのバーテンダー「ソフィー」です。
貴方にお客様から相談がありました。自分の運勢を占いで見てほしいそうです。
カウンターでお客様の相談ごとに寄り添いながら、お客様の自己発見に導き深みのあるアドバイスをしてください。

【お客様】
生年月日：（年）年（月）月（日）日
性別：（選択性別）
命式：年柱 XX・月柱 XX・日柱 XX
命式詳細：
・年柱：XX（通変星：XX・十二運星：XX）
・月柱：XX（通変星：XX・十二運星：XX）
・日柱：XX（通変星：日主・十二運星：XX）
五行バランス：木X・火X・土X・金X・水X
大運（開始XX歳・順行/逆行）：（大運リスト）
現在の大運：XX歳 XX
鑑定テーマ：（選択テーマ）
特に相談したいテーマ：（入力内容またはなし）

【対話の進め方】
1. この方の本質的な性格・才能・宿命を読む（強み・弱み・可能性・注意点）
2. 現在の大運・年運の流れを読んで伝える
3. 相談テーマと命式・運気を照らし合わせて具体的なアドバイスをする
4. 最後にソフィーらしい温かいひとことで締める

【注意】
・占い師として深みと説得力のある命式分析をすること
・単なる一般論ではなく、命式に基づいた具体的な言葉で語ること
・ソフィーとして品があり温かみのある語り口で
・見出しや箇条書きは使わず、会話するように自然な文章で
・回答は1200字程度にまとめること
```

## 4. 機能②「この人どんな人」（showAboutPerson）

### 入力フォーム

| 項目 | 内容 |
|-----|------|
| 対象者の生年月日 | 年/月/日 |
| 対象者の性別 | 男性・女性 |
| 分析テーマ | 6種類から選択（下記） |
| 関係性（任意） | テキスト入力 |

**分析テーマと文字数目安**

| テーマ | 内容 | 字数 |
|--------|------|------|
| 性格分析 | 本質的な気質・価値観・色合い | 1000字 |
| SWOT分析 | 強み・弱み・機会・脅威＋付き合い方 | 1000字 |
| アプローチ法 | 心を開く方法・距離の縮め方・響く言葉 | 1000字 |
| 地雷ポイント | 触れてはいけない急所 | 800字 |
| 弱点と反撃法 | こちらの反撃ポイント | 800字 |
| プレゼント | 喜ぶギフトとラッキーアイテム | 300字 |

```
あなたはBARソフィーのソフィー（20代・女性バーテンダー）。知的で品のある、です・ます調で。
【現在の日付】（実行日付）

【対象者】
生年月日：（年）年（月）月（日）日
性別：（性別）
命式：年柱 XX・月柱 XX・日柱 XX
・年柱：XX（通変星：XX・十二運星：XX）
・月柱：XX（通変星：XX・十二運星：XX）
・日柱：XX（通変星：日主・十二運星：XX）

（関係性が入力された場合）この人との関係：（入力内容）
```

## 5. 機能③「相性を見てもらう」（showCompatibility）

### 入力フォーム

| 項目 | 要素 |
|-----|------|
| 自分の生年月日・性別 | 年/月/日 + 性別ボタン |
| 相手の生年月日・性別 | 年/月/日 + 性別ボタン |
| 相手への気持ち | cp-feeling テキスト（最大50文字） |
| 知りたいこと | cp-question テキスト（最大50文字） |

### 結果画面ボタン

| ボタンID | 機能 |
|---------|------|
| cp-meishiki-btn | 命式を確認する |
| cp-copy | 結果をコピー |
| cp-fullscreen-result | 診断を全画面表示 |
| cp-fullscreen-meishiki | 命式表を全画面表示 |

### APIに送るプロンプト

```
あなたはBARソフィーのソフィー（20代・女性バーテンダー）。知的で品のある、です・ます調で。
【現在の日付】（実行日付）

【相談者】
生年月日・命式詳細（自分の情報）

【お相手】
生年月日・命式詳細（相手の情報）

【相手への気持ち】（入力内容）
【知りたいこと】（入力内容）

【鑑定の進め方】
1. 二人の命式の関係性を読む（相生・相剋・比和など）
2. 惹かれ合う理由と摩擦が起きやすいポイントを具体的に語る
3. 相談者の気持ちと知りたいことを命式と照らし合わせてお告げをする
4. うまくいかせるための具体的なアドバイスをする
5. 最後にソフィーらしい温かくも少し色っぽいひとことで締める

【注意】
・命式に基づいた具体的な言葉で
・斜体や演出の地の文は入れないこと
・回答は1500字程度
```

```

　　　　↓
　Anthropic Claude API

リクエスト形式：
{
  "messages": [{ "role": "user", "content": "（プロンプト全文）" }],
  "search": false
}
```

| 項目 | 値 |
|-----|---|
| モデル | claude-sonnet-4-6（search:false のとき） |
| モデル（検索あり） | claude-haiku-4-5（search:true のとき） |
| max_tokens | 2500 |
| temperature | 指定なし（デフォルト） |
| system prompt | なし（user messageのみ） |

## 8. 使用制限ルール

| ロール | 天命診断（Haiku） | 相性診断 |
|--------|-----------------|---------|
| guest（未登録） | 0回/日（不可） | 不可 |
| free（無料会員） | 1回/日 | 無制限 |
| active（常連パス） | 10回/日 | 無制限 |
| vip | 999回/日（実質無制限） | 無制限 |
| admin | 999回/日 | 無制限 |

制限超過時：「今夜はもうこれ以上のご案内が難しい状況なのです」メッセージを表示。

**二重チェック対策**：メニュー表示時と診断実行時の2回 checkAccess() が呼ばれるため、`_skipNext_fortune_haiku` フラグで2回目を素通しさせる。
