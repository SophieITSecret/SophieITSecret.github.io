## 3. ライト/プレミア グレード体系

### 3-1. グレード定義

| グレード | モデル | アクセス制御 | 用途 |
|---------|--------|------------|------|
| **ライト診断** | Claude Haiku 4.5 | `fortune_haiku`（日次カウンタ） | 100字以内の短いお告げ |
| **プレミア診断** | Claude Sonnet 4.6 | `fortune_sonnet`（プレミアチケット消費） | 詳細な命式分析 |

### 3-2. ライトチケット（日次回数）

| ロール | 1日あたりの上限 | 補足 |
|--------|--------------|------|
| guest（未登録） | 0回（不可） | ログインプロンプト表示 |
| free（無料会員） | 1回/日 | |
| active（常連パス） | 3回/日 | |
| vip | 999回（実質無制限） | |
| admin | 999回（実質無制限） | |

- Firestore の `lastFortuneBDate` / `dailyFortuneBCount` で管理
- 日付変更でリセット
- UIの●ドット表示は最大5つ（5以上あっても5つ全点灯）

### 3-3. プレミアチケット（旧Sチケット）

| ロール | 入手方法 |
|--------|---------|
| free / active | 毎日2枚・毎週5枚 自動配給 ＋ 購入可 |
| vip / admin | 無制限（チケット消費なし） |

- Firestore の `dailyTickets` / `weeklyTickets` / `purchasedTickets` の合計
- 合計 = プレミアチケット残数
- UIの●ドット表示は最大5つ

### 3-4. ドット表示計算（共通）

```javascript
const makeDots = (filled, total) => Array.from({length: total}, (_, i) =>
    `<span style="color:${i < filled ? '#f0b56e' : '#444'}; font-size:0.75rem;">●</span>`).join('');

const lightDotsHtml = makeDots(Math.min(5, lightRemaining), Math.min(5, lightLimit));
const premiumDotsHtml = makeDots(Math.min(5, totalTickets), 5);
```

## 4. GAS へのリクエスト構造

```json
{
  "messages": [{ "role": "user", "content": "（プロンプト全文）" }],
  "search": false,
  "grade": "light" | "premium"
}
```

| パラメータ | 値 |
|----------|---|
| エンドポイント | Google Apps Script (GAS) 経由 → Anthropic API |
| `grade: "light"` → モデル | `claude-haiku-4-5-20251001` |
| `grade: "premium"` → モデル | `claude-sonnet-4-6` |
| `search: false` | 常にfalse（天命診断では検索不使用） |
| max_tokens | GAS側で設定（ライト: 512程度 / プレミア: 2048程度） |
| system prompt | なし（user messageのみ） |

## 6. 機能① 今日のアドバイス（showDailyAdvice）

### 入力フォーム

| 項目 | 要素 |
|-----|------|
| グレード選択 | ライト診断 / プレミア診断（ドット表示付き） |
| 生年月日 | da-year / da-month / da-day |
| 性別 | da-gender ボタン（男・女） |
| 人物帳 | da-people ボタン（別の人のデータも選択可） |

- デフォルトは `bar_sophie_self`（localStorage）から自分の生年月日を自動入力
- 人物帳から他の人のデータも読み込み可能

### キャッシュ仕様

- 送信後の結果を `sophie_daily_fortune` キー（localStorage）に保存
- 保存形式：`{ date, text, grade, birth, gender }`
- 翌日になるまで同じ結果を表示（「本日取得済み」ラベル付き）
- 「もう一度引き直す」ボタンでキャッシュを削除→再取得可（ライト/プレミア再消費）
- カウンターのピルボタン表示もキャッシュ有無で色が変化（緑✓ / グレー）

### アクセス制御

| グレード | checkAccess |
|---------|-------------|
| ライト | `fortune_haiku`（日次カウンタ） |
| プレミア | `fortune_sonnet`（プレミアチケット消費） |

---

## 14. 結果テキストの整形（共通）

```javascript
resultText
    .replace(/\*\*(.+?)\*\*/g, '<strong>$1</strong>')
    .replace(/\*([^*]+)\*/g, '')            // 斜体マークダウン除去
    .replace(/\*\*/g, '')
    .replace(/^#{1,3}\s*(.+)$/gm, '<span style="color:#f0b56e; ...">$1</span>')
    .replace(/^-{2,}$/gm, '')              // 水平線除去
    .replace(/\n{2,}/g, '\n')
    .replace(/\n/g, '<br>')
```
